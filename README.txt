Descrição do projeto:

Este projeto foi desenvolvido a partir de um mock recebido pela empresa Linx.
As implementações foram realizadas em HTML5, CSS3 e JavaScript. Sem uso de 
framework Bootstrap. 

O projeto tem por base uma grade de produtos com imagem, nome, descrição e seus respectivos valores.
Foi utilizado de uma API fornecida pela empresa para puxar as informações dos produtos. Porém, tive 
dificuldades em colocá-los lado a lado, no que decidi apenas trazer a informação de um dos produtos 
que estavam no array, desta forma pude mostrar que fui capaz de acessar os dados da lista.

Também consegui implementar a validação dos campos "Nome" e "E-mail", mostrando em tela uma mensagem em 
vermelho caso os campos estejam incorretos e uma em verde para o caso destes estarem corretos.

Tratei de manter-me o mais fiel possível às medidas fornecidas, no entando em alguns casos precisei
adaptar algumas destas, já que não se comportaram como o esperado.

Decidi priorizar os pontos onde percebi que poderia trazer um resultado positivo com mais segurança e rapidez.
Por outro lado, tentei implementar a função de carregar novos dados na mesma página com 8 novos produtos, mas
não fui capaz de fazer funcionar a função dentro do prazo.

Apesar do curto período de tempo, pude aprender muitas funcionalidades novas com este projeto.
Continuarei trabalhando nele até aperfeiçoa-lo da forma que gostaria. Foi uma experiência incrível e desafiadora
haver participado desta seleção e fico muito agradecida de ter tido esta oportunidade.
